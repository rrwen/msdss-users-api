    • load_env (bool) – Whether to load variables from a file with environmental variables at env_file or not.

    • driver_key (str) – The environmental variable name for driver.

    • database_key (str) – The environmental variable name for database.

    • engine_kwargs (dict) – Keyword arguments passed to msdss_base_database.core.Database if engine is None.

    • async_database_kwargs (dict) – Keyword arguments passed to databases:databases.Database if async_database is None.

    • Function for the user database dependency from get_user_db or auto-configured msdss_users_api.tools.create_user_db_func().

    • Async database from parameter async_database or auto-configured databases:databases.Database from env vars

    • Engine from parameter engine or auto-configured sqlalchemy.engine.Engine from env vars

    Return type

    Author

    Example

    import databases

from msdss_base_database import Database
from msdss_users_api.tools import create_user_db_func

    Create a function to return the the user manager class.

    See UserManager configuration for FastAPI Users,

    Parameters

    • get_user_db (func) – Function for the user database dependency. See msdss_users_api.tools.create_user_db_func().

    • user_manager_model (msdss_users_api.models.UserManager) – The user manager model from FastAPI Users. See msdss_users_api.models.UserManager and msdss_users_api.tools.create_user_manager_model().

    Returns

    A function yielding a configured msdss_users_api.models.UserManager.

    Return type

    func

    Author

    Richard Wen <rrwen.dev@gmail.com>

    Example

    import databases

from msdss_base_database import Database
from msdss_users_api.tools import *

# Create a database object
db = Database()

# Get the engine object from the database
engine = db._connection

# Get the async database
async_database = databases.Database(str(engine.url))

# Get the user db func
get_user_db = create_user_db_func(engine, async_database)

# Create a user manager model
UserManager = create_user_manager_model('msdss-secret')

# Get the user manager func
get_user_manager = create_user_manager_func(get_user_db, UserManager)
